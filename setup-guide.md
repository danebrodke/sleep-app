# What to Install (Mac)

1. **Developer Tools** — Open Terminal, run `xcode-select --install` — Gives you Git, which tracks your code changes
2. **Node.js** (nodejs.org, LTS version) — Runs web apps on your computer
3. **Python** (python.org) — Lets AI do data work, file processing, and automation
4. **VS Code** (code.visualstudio.com) — Where you see and edit your code
5. **Claude Code extension** (inside VS Code) — The AI that builds your app for you
